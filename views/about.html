About this site

Everyone should have the chance to vote, wether they are able to visit the polling station or not. Unfortunately it isn't possible to apply for a postal vote online, so I decided to build something to make the process a little easier.

All code is available on GitHub, pull requests graciously accepted as there are still a bunch of rough edges!

There are a number of features still to complete:
